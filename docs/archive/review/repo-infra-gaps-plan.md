# Plan: repo-infra-gaps

## Objective

Fill missing repository infrastructure files identified as gaps after the security review improvements PR.

## Requirements

1. `.github/dependabot.yml` — Keep GitHub Actions dependencies up to date automatically
2. `CHANGELOG.md` — Document version 1.0.0 changes (was in the original plan but omitted)
3. `.gitignore` — Add `smart-ssh.sha256` and `smart-ssh.sha256.asc` (generated by `make checksum`/`make release`)

## Technical Approach

### 1. dependabot.yml
- Monitor `github-actions` ecosystem only (no npm/pip/etc since this is a shell project)
- Weekly schedule
- Target branch: main

### 2. CHANGELOG.md
- Follow [Keep a Changelog](https://keepachangelog.com/) format
- Document 1.0.0 as the initial release with all current features
- Group changes into Added/Changed/Security sections

### 3. .gitignore
- Append `smart-ssh.sha256` and `smart-ssh.sha256.asc` to existing .gitignore

## Implementation Steps

1. Create `.github/dependabot.yml`
2. Create `CHANGELOG.md`
3. Append to `.gitignore`
4. Run tests to verify no regressions

## Testing Strategy

- Existing bats tests must still pass (no code changes)
- Verify dependabot.yml syntax with `actionlint` or manual inspection
- Verify .gitignore correctly ignores checksum files

## Considerations & Constraints

- CHANGELOG.md should be English only (per project convention)
- dependabot.yml only covers github-actions (no other package ecosystems)
